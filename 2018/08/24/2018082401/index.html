xadmin源码目录简说

  • locale 对应语言包
  • migrations 是数据表的记录
  • plugins 每一个后台页面都是一个plugin插件
  • static 静态文件夹，里面有js,css
  • template 这是xadmin自己使用的html文件
  • templatetags 这是tag模板
    总而言之，xadmin它是对Django admin的封装和进阶开发，正所谓青出于蓝而胜于蓝就是这个理。
